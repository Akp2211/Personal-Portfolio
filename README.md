# Personal Portfolio

A modern and responsive portfolio website showcasing my projects, technical skills, achievements, and experience as a Computer Science Engineering student.

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* Responsive Design
* Git & GitHub

## Features

* Responsive UI for desktop and mobile devices
* Project showcase section
* Skills and technology stack overview
* Education and achievements section
* Contact and social links
* Smooth navigation and modern design

## Featured Projects

### Shiksha DC — Distributed Classroom Content Hub

* Built a distributed classroom platform using FastAPI, Redis, Kafka, CockroachDB, and MinIO.
* Implemented JWT authentication, RBAC, and Redis-backed session management.
* Designed fault-tolerant storage and high-availability architecture.

### Cricket Scoring Website

* Developed a real-time cricket scoring platform for 20+ teams.
* Integrated Firebase Realtime Database for live score synchronization.
* Built responsive UI components using Next.js and Tailwind CSS.

### Coding Adda

* Full-stack e-learning platform for educational content delivery.
* Developed using React, Node.js, SQL, HTML, CSS, and JavaScript.

## Local Setup

```bash
git clone https://github.com/Akp2211/Personal-Portfolio.git
cd Personal-Portfolio
```

Open `index.html` in your browser or run using a local development server.
